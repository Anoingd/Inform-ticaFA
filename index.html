Hardware

O hardware contempla toda a parte física do computador. Podemos fixar esta ideia na cabeça pensando nesta pequena frase: Hardware é aquilo que chutamos quando o computador trava.

Alguns exemplos do que seria o hardware: monitor, teclado, mouse, placa-mãe, memória RAM, disco rígido. Se você olhar para o seu celular, toda a parte física que você toca é considerada hardware!

Classificação:

  • Entrada → teclado, mouse, microfone, câmera.
  • Saída → monitor, caixas de som, impressora.
  • Entrada e Saída → tela touchscreen, pendrive, HD externo.

Componentes internos de um computador:

  • Placa-mãe → “a central que conecta todas as peças”.
  • Processador (CPU) → “o cérebro do computador, onde os cálculos acontecem”.
  • Memória ram → “memória temporária, usada enquanto o PC está ligado”.
  • Disco rígido / SSD → “onde ficam armazenados arquivos e programas”.
  • Fonte de alimentação → “leva energia para todas as peças”.
  • Placa de vídeo (GPU) → “responsável pelas imagens e jogos”.

Software

O software é a parte lógica, ou seja, os programas que rodam no computador. Exemplos: Windows, Word, navegadores de internet e jogos.

Alguns exemplos do que seria um software:

Internet

A internet conecta computadores do mundo todo. Com ela podemos enviar mensagens, estudar, trabalhar e acessar informações.

Office

O Pacote Office foi lançado para Windows em 1990, ele é sem duvidas a suíte mais usada para trabalhos em todo o mundo, sendo essencial para basicamente todo trabalho existente.
